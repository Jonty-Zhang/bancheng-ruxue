# 伴城入学 · 随迁子女入学政策智能助手

第十一届中国数据新闻大赛暨第二届 AIGC 应用大赛参赛作品。

家长选择城市、填几项家庭情况，应用基于 8 座流入城市（北京 / 上海 / 广州 / 深圳 / 苏州 / 成都 / 东莞 / 武汉）
的官方政策**实时推演**：能不能入学、中考能不能上普高、要准备哪些材料，并由大模型生成**下一步操作指南、
咨询话术、智能问答**。每条结论都挂官方政策原文来源。

## 技术架构

```
政策库（人工核对，挂官方来源）
        │
   规则引擎（前端 JS，确定性、零幻觉）──→ 入学/中考判断、材料清单、八城阶梯
        │
   后端 /api/chat（server.py，安全代理 DeepSeek）──→ 操作指南 / 咨询话术 / 智能问答
        │
   大模型只做“说人话、列步骤、提问题”，全部基于政策库，标来源、可追溯
```

- `index.html` `style.css` `app.js` — 前端（Apple 风格 + 规则引擎 + 政策库）
- `server.py` — 轻量后端（纯 Python 标准库，零第三方依赖）：托管网页 + 安全代理 DeepSeek（API key 不进前端）

## 运行方式

**方式一：完整体验（含 AI 实时生成）**
```bash
cd 伴城入学
DEEPSEEK_API_KEY=你的key python3 server.py
# 浏览器打开 http://localhost:8000
```

**方式二：快速预览（无 AI key 也能跑）**
- 直接双击 `index.html`，或 `python3 server.py` 后打开 http://localhost:8000
- 规则推演、材料清单、八城阶梯完全可用；AI 区走**离线兜底示例**（基于政策库的模板）

## AI 使用与合规

- 大模型仅基于**人工核对的官方政策库**作答，不自由发挥、不编造政策；
- 每条建议提示“以当地教育部门当年官方文件为准”，并附政策原文链接；
- API key 由后端持有、不暴露于前端，符合“真实性、可解释性、合规”要求。

## 数据来源

8 城教育局 / 招生考试机构 / 政府门户公开文件；全国背景数据来自国家统计局 / UNICEF / UNFPA、教育部；
政策背景为国务院《关于推行常住地提供基本公共服务的实施意见》（2026.05.22）。
